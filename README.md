# PaperGrab

> Find academic papers, open-access PDFs, and citations using natural language.

## Introduction

PaperGrab is an AI-powered academic paper discovery tool that helps researchers find relevant papers using plain English.

Instead of manually searching across Google Scholar, Semantic Scholar, arXiv, OpenAlex, and other academic databases, simply describe your research topic and PaperGrab will automatically discover relevant papers, prioritize open-access PDFs, and generate formatted citations.

PaperGrab is designed to make literature review faster, easier, and more accessible for students, researchers, and professionals.

---

# Why Use PaperGrab?

## The Problem

Finding academic papers is often time-consuming:

- Searching across multiple academic databases
- Experimenting with different keywords repeatedly
- Filtering irrelevant results manually
- Looking for accessible PDF versions
- Formatting references afterward

Many researchers spend more time finding papers than reading them.

## What PaperGrab Solves

PaperGrab helps you:

- Search papers using natural language
- Discover papers from multiple scholarly sources simultaneously
- Prioritize open-access papers with downloadable PDFs
- Generate citation-ready references automatically
- Accelerate literature reviews and research workflows

# How It Works

1. Describe your research topic in natural language
2. AI converts your request into scholarly search queries
3. PaperGrab searches multiple academic databases
4. Results are deduplicated and ranked
5. Open-access PDF links and formatted citations are returned

---

# Technical Highlights

PaperGrab follows a token-efficient research workflow:

- AI is used only for search planning
- Scholarly APIs are queried directly
- Compact metadata is retrieved instead of full webpage content
- Results are deduplicated and ranked programmatically
- Open-access PDFs are prioritized whenever available

This approach improves search speed, reduces token usage, and lowers operational costs.

# 论文寻找好帮手

> 用自然语言快速找到学术论文、开放访问 PDF 和参考文献。

## 项目简介

PaperGrab 是一个 AI 驱动的学术论文搜索工具，帮助研究人员通过自然语言快速发现相关论文。

无需在 Google Scholar、Semantic Scholar、arXiv、OpenAlex 等多个学术数据库之间来回切换，只需描述你的研究主题，PaperGrab 就会自动搜索相关论文、优先返回开放访问（Open Access）PDF，并生成格式化参考文献。

PaperGrab 致力于帮助学生、研究人员和专业人士更高效地完成文献调研工作。

# 为什么使用 PaperGrab？

## 传统论文搜索的痛点

寻找论文通常需要：

- 在多个学术数据库之间切换
- 不断尝试不同关键词
- 手动筛选无关结果

- 后续整理参考文献格式

PaperGrab 可以帮助你：

- 自动生成可直接引用的参考文献
- 提升文献调研效率

# 工作原理

1. 使用自然语言描述研究主题
2. AI 自动生成学术搜索关键词
3. 搜索多个学术数据库
4. 对结果进行去重与排序
5. 返回 PDF 链接和格式化参考文献

---

# 技术特点

PaperGrab 采用 Token 高效的研究工作流：

- AI 仅负责搜索规划
- 直接调用学术数据库 API
- 获取精简元数据而非完整网页内容
- 使用程序完成去重与排序
- 优先返回开放访问 PDF

这种设计能够降低 Token 消耗，同时提升搜索速度和搜索质量。
